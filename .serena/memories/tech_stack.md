# Technology Stack

## Programming Languages
- **Python 3.x** (Primary) - All automation scripts
- **Bash/Shell** - System administration and Docker management
- **YAML** - Configuration files (backup, update configurations)

## Core Technologies
- **Docker & Docker Compose** - Container orchestration
- **Nginx** - Reverse proxy and web server
- **Let's Encrypt** - SSL/TLS certificate management
- **7-Zip/zstd** - Compression formats for backups

## Key Dependencies
- **Python Libraries**: 
  - Standard library (subprocess, json, yaml, logging)
  - Third-party: requests, packaging (for version management)
- **System Tools**:
  - Docker engine
  - Nginx
  - Various compression tools (7z, zstd, gzip)
  - System package managers (apt)

## Infrastructure
- **Git** - Version control with branch-based releases
- **ZSH + Oh My Zsh** - Enhanced shell environment
- **Fail2ban** - Security management
- **systemd** - Service management

## Platform Support
- **Primary**: Ubuntu/Debian Linux
- **Secondary**: macOS (Darwin) with compatibility adjustments
